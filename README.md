# 🎁 Challenge Amigo Secreto  

## 🎯 Objetivo  
Desarrollar una aplicación interactiva que permita ingresar nombres y realizar el sorteo de un "amigo secreto".  

## 🛠️ Funcionalidades  
✔️ **Agregar nombres**: Los usuarios pueden ingresar nombres a través de un campo de texto y un botón.  
✔️ **Validación**: Se evita el ingreso de nombres vacíos o duplicados.  
✔️ **Visualización**: La lista de participantes se muestra en pantalla.  
✔️ **Sorteo aleatorio**: Se asignan parejas de "amigo secreto" de manera aleatoria y se muestra el resultado.
